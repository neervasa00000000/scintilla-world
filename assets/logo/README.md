# Logo & Favicon Files

## Favicon Setup

The favicon is referenced as `/assets/logo/favicon.png` across all HTML files.

**To update the favicon:**
1. Save your favicon image as `favicon.png` in this folder (`assets/logo/`)
2. Recommended size: 32x32px or 64x64px (PNG format)
3. The favicon will automatically appear on all pages

## Current Files

- `favicon.png` - Main favicon (save your stylized "S" image here)
- `scintilla.png` - Previous favicon (can be removed)
