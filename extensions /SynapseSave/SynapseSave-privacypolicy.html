Summary: SynapseSave does not collect, transmit, or store any user data on external servers. All data is stored locally on your device.

1. Data Collection

SynapseSave does not collect any user data.

We do not collect, transmit, or store:

  • Personal information (name, email, address, etc.)
  • Browsing history
  • Website content
  • Location data
  • Authentication credentials
  • Financial information
  • Any other personal or sensitive data

2. Local Storage

SynapseSave uses Chrome's local storage API to store information about snoozed tabs locally on your device. This includes:

  • Tab URLs
  • Tab titles
  • Scheduled wake times
  • Tab group associations

This data never leaves your device. It is stored locally in your browser and is not transmitted to any external servers or third parties.

3. Data Usage

The locally stored data is used solely for:

  • Remembering which tabs you've snoozed
  • Scheduling when tabs should reopen
  • Restoring tabs to their original browser groups
  • Displaying statistics (tabs snoozed, RAM saved, etc.)

This data is not used for any other purpose, including advertising, analytics, or tracking.

4. Third-Party Services

SynapseSave does not use any third-party services, analytics tools, or external APIs. The extension operates entirely within your browser using standard Chrome extension APIs.

5. Permissions

SynapseSave requires the following permissions:

  • tabs: To access tab information, close tabs when snoozed, and reopen them later
  • tabGroups: To restore tabs to their original browser groups
  • storage: To store snoozed tab information locally on your device
  • alarms: To schedule when snoozed tabs should wake up
  • notifications: To notify you when snoozed tabs are ready to reopen
  • favicon: To display website icons in the extension interface

All permissions are used solely for the core functionality of the extension and are not used to collect or transmit data.

6. Data Sharing

We do not share, sell, or transfer any data to third parties. Since we don't collect any data, there is nothing to share.

7. Data Security

All data stored by SynapseSave is stored locally on your device using Chrome's secure local storage API. This data is protected by your browser's security measures and is not accessible to external parties.

8. User Control

You have full control over your data:

  • You can delete snoozed tabs at any time through the extension interface
  • You can uninstall the extension, which will remove all locally stored data
  • You can clear browser data, which will remove extension storage

9. Children's Privacy

SynapseSave does not knowingly collect any information from children. Since we don't collect any data, this is not applicable.

10. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date. Since we don't collect data, any future changes would only be to clarify our practices.

11. Compliance

SynapseSave complies with:

  • Chrome Web Store Developer Programme Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)

Since we don't collect any data, compliance is straightforward - we simply don't collect, use, or share any user data.
